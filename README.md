-> Project 4 (https://github.com/ANMOL0me/DSA_in_java/blob/main/Advance_of_Parking_System.java) :-Advance_of_Parking_System program is a more advanced version of a parking_system that uses a Doubly Linked List with a HashMap to efficiently manage cars entering and leaving the parking area from different positions. Each car is represented by a Node containing the car number and pointers to the previous and next cars, allowing movement in both directions. The ParkingSystem class maintains pointers to the front, rear, and two middle positions (mid1 and mid2) of the parking list. Cars can enter from the front, rear, or middle (middle entry requires special permission). When a car is added, a new node is created and linked appropriately, the car is stored in a HashMap (carMap) for fast access, and the middle pointers are updated based on the current size of the list. The removeCar method allows direct removal of a specific car by its number without searching the entire list by using the HashMap to locate its node instantly and reconnect the neighboring nodes. The display method traverses the list from front to rear and prints the parked cars. The main advancement of this system compared to a basic queue-based parking system is that it supports multiple entry points (front, rear, middle), direct car exit without disturbing other cars, constant-time car lookup using HashMap, and efficient insertion/removal using a doubly linked list, making the parking system more flexible and closer to a real-world smart parking management model.

-> Project 5 (https://github.com/ANMOL0me/DSA_in_java/blob/main/LINKLIST_in_STACK_in_QUEUE.java) :- This program builds a nested data structure using concepts from Data Structures and Algorithms where a Linked List stores nodes, each node contains a Stack, and each stack element is a Queue . The Node class represents a linked list node that holds a stack (Stack<Queue<Integer>>) and a pointer to the next node. The LinkedListSystem class manages the structure: addNode() creates new nodes in the linked list, addQueueToStack() pushes a queue into the stack of a specific node, and display() traverses the linked list and prints all queues stored inside each node’s stack. In main(), two nodes are created, two queues with numbers are added to the stack of the first node, and finally the entire structure is displayed, demonstrating how multiple data structures (linked list → stack → queue) can be combined in one system.
