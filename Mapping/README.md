# Smart Route Finder 🗺️

A web app that finds shortest path using Dijkstra Algorithm.

## Tech Stack
- Java (Spring Boot)
- HTML, CSS, JavaScript

## Features
- Find shortest path between cities
- REST API backend
- Interactive UI

## How to Run
1. Run backend (Spring Boot)
2. Open frontend/index.html
3. Enter cities

## Example
Delhi → Mumbai  
Distance: 1440  
Path: Delhi → Jaipur → Udaipur → Mumbai
